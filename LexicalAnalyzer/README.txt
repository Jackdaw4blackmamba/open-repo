簡易的な字句解析プログラム

***以下の書籍を大いに参考にしています***
林晴比古, 「明快入門 コンパイラインタプリタ開発」, ソフトバンク クリエイティブ株式会社, 第1刷. 
を参考に, トークンとして分けられる範囲を拡大. また, ファイルからの読み込みだけでなくchar配列を直接渡しても解析可能.
これにより, あるコードの一部だけを解析することなども可能. 対象言語はC.

使用例:
> gcc main.c token_funcs.c -o main
> main token_funcs.c

テスト用のメインではコマンドライン引数で受け取ったファイルを読み込んで解析を行います.
